Relative: A1

Changes when copied

Most common type

Absolute: $A$1

Never changes

For fixed values

Mixed: $A1 or A$1

Partially fixed

Advanced use cases

26 / 37

Mean (Average)

Excel: =AVERAGE(range) Formula: Sum of all values ÷ Number of values

✅ Strengths

  • Uses all data points
  • Mathematically robust
  • Good for normal data

❌ Weaknesses

  • Sensitive to outliers
  • Can be misleading
  • Not for skewed data
27 / 37

Median

Excel: =MEDIAN(range) Definition: Middle value when data is sorted

✅ Strengths

  • Robust to outliers
  • Better for skewed data
  • Represents "typical"

❌ Weaknesses

  • Ignores data magnitude
  • Less mathematical
  • Can miss patterns
28 / 37

Mode

Excel: =MODE.SNGL(range) Definition: Most frequently occurring value

✅ Strengths

  • Works with categories
  • Finds popular choice
  • Easy to understand

⚠️ Considerations

  • May not exist
  • Multiple modes possible
  • Less informative
29 / 37

Choosing the Right Measure

Situation Best Measure Why?
Normal distribution Mean Uses all information
Skewed data / Outliers Median Not affected by extremes
Categorical data Mode Only option for categories
Income data Median Few high earners skew mean
